# stat 모델 적용 테스트 결과

## 1. 목적

통계분석 모듈의 키워드 및 연관어 품질을 개선하기 위해 파운데이션 모델과 임베딩 기반 모델을 선택적으로 적용할 수 있는 구조를 구현했다.

---

## 2. 적용 모델

| 기능 | 모델 / 도구 | 설명 |
|---|---|---|
| 핵심 키워드 후보 추출 | `Qwen/Qwen2.5-1.5B-Instruct` | 한국어 뉴스 본문에서 핵심 키워드 후보를 JSON 형식으로 추출 |
| 연관어 후보 추출 | `KeyBERT` + `jhgan/ko-sroberta-multitask` | 문서 임베딩 기반으로 본문과 관련성이 높은 단어 또는 구를 추출 |
| 수치 계산 | Python | 키워드 등장 횟수, 문장 수, 단어 수, 월별 언급량 계산 |

---
